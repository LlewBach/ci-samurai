Code Eye Samurai

2D side scroll fighting game

In 2024...

The Shogun of Code Institute Dojo, had set an assignment to dominate JavaScript and return with honour.

One coder, suffering defeats from the smallest bugs, closed himself off from the outside world...

completely.

Normal people would never see his face...

for he only had eyes for a sweet-ass codebase.

For at that time, code eyes were considered sub-human...

Pitiable, bug-infested, obsessed.

From this era arose a legend

of a coder,

of a debugging sesh,

of revenge.

The Code Eye Samurai

Keyboard

Spacebar - Start/Pause

- Reset Game

- Annotate Mode

- Run Left

- Run Right

- Jump

- Hold to roll

- Attack 1

- Attack 2

- Attack 3

If standing, hold Shift to attack left

Touchscreen

Swipe up to Start/Pause

Swipe down to Enter/Exit Training Mode or restart game

Joystick to move, buttons to attack

If standing, hold joystick button to attack left

How to play

if (you need energy) {jump to generate}

else if (you are being hit) {roll out of it}

else if (you are being running) {you don't need to hold shift to attack left}

else {kick some ass}

if (score === 333) {chosenOne = true;}
